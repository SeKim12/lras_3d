## Usage
This repo provides the example codes for Novel View Synthesis, 3D Object Manipulation, and Depth Estimation.

### Novel View Synthesis
```bash
python run_nvs.py
```

### 3D Object Manipulation
```bash
python run_object_motion.py
```

### Depth Estimation
```bash
python run_depth.py
```
